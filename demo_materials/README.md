# 데모 자료

SPEC Agent 화면에서 바로 업로드해 볼 수 있는 예시 입력 자료입니다.

## 사용 방법

1. 웹 화면에서 `파일` 또는 `폴더` 버튼을 누릅니다.
2. 아래 폴더 중 하나의 파일을 모두 업로드합니다.
3. 채팅창에 아래 문장을 보냅니다.

```text
첨부 자료를 분석해서 출원명세서 검토용 초안을 작성해줘.
부족한 항목은 질문으로 알려주고, 근거 없는 수치나 도면부호는 만들지 마.
```

## 폴더별 용도

| 폴더 | 주제 | 확인하기 좋은 점 |
|---|---|---|
| [`complete_baby_chair_case`](complete_baby_chair_case) | 슬라이드식 보조받침 유아용 의자 | 도면부호와 구성요소가 있는 제품 발명 |
| [`complete_flying_car_case`](complete_flying_car_case) | 접이식 덕트팬 비행 자동차 | 넓은 아이디어가 구체 자료로 바뀌는 흐름 |
| [`food_shape_case`](food_shape_case) | 층상 식품 성형체 | 식품 구조, 재료 후보, 제조 흐름 |
| [`ergonomic_chair_case`](ergonomic_chair_case) | 자세 보조 인체공학 의자 | 센서, 조절부, 알림 구조 |
| [`ai_software_case`](ai_software_case) | AI 소프트웨어 발명 | 데이터 처리 흐름과 소프트웨어 구성 |
| [`complete_case`](complete_case) | 회전각 고정 레버 장치 | Markdown 파일 기반 기본 완성 예시 |

각 폴더의 `README.md`에는 파일 구성과 테스트 포인트를 따로 적어 두었습니다.
