# 발표용 입력 자료 모음

각 폴더를 SPEC Agent 화면에서 폴더 업로드로 올린 뒤 아래 문장을 보내면 됩니다.

```text
첨부 자료를 모두 분석해서 출원명세서 검토용 초안을 작성해줘.
부족한 항목이 있으면 답장 안에서 먼저 질문하고, 오른쪽 체크리스트에도 표시해줘.
자료에 없는 실험 수치, 문헌번호, 효과는 만들지 말고 검토 필요 항목으로 남겨줘.
```

## 폴더

- `complete_case/`: 회전각 고정 레버 장치 기본 데모
- `complete_flying_car_case/`: 질문 없는 완성형 데모 1, 접이식 덕트팬 비행 자동차
- `complete_baby_chair_case/`: 질문 없는 완성형 데모 2, 슬라이드식 보조받침 유아용 의자
- `food_shape_case/`: 새로운 음식 형태 특허 데모
- `ergonomic_chair_case/`: 인체공학 의자 특허 데모
- `ai_software_case/`: AI 소프트웨어 개발 특허 데모

`complete_flying_car_case/`와 `complete_baby_chair_case/`는 처음부터 필수항목 10개가 모두 들어 있는 완성형 시나리오입니다. 질문이 거의 없이 초안과 Word 산출물로 넘어가는 상황을 보여줄 때 사용합니다.

새로 추가한 주제별 데모 폴더는 대부분 `.txt` 파일 4개로 구성되어 있습니다. 회의록/상담기록, 구성요소, 도면부호, 선행기술 키워드, 효과, 산업상 이용가능성을 나누어 담았습니다.
