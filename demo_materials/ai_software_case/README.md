# AI 소프트웨어 발명

사용자 입력 자료를 분석하고 결과를 추천·분류·생성하는 AI 소프트웨어 발명 예시입니다.

## 파일 구성

- `01_회의녹취록.txt`: 서비스 문제와 사용 장면
- `02_아이디어_구성작동.txt`: 입력 데이터, 전처리, 임베딩, 검색, 생성, 검증 흐름
- `03_도면부호_설명.txt`: 시스템 구성도와 모듈 부호
- `04_선행기술_효과_산업상이용가능성.txt`: 비교 키워드, 정성 효과, 적용 분야

## 테스트 포인트

- 소프트웨어 발명도 장치/방법 형식으로 구조화되는지 확인
- 데이터 처리 단계가 작동 방식/실시예로 정리되는지 확인
- 효과가 과장된 성능 수치가 아니라 정성 효과로 남는지 확인

## 권장 메시지

```text
첨부 자료를 분석해서 AI 소프트웨어 발명의 출원명세서 검토용 초안을 작성해줘.
자료에 없는 정확도나 성능 수치는 만들지 마.
```
