# 회전각 고정 레버 장치

Markdown 파일로 구성된 기본 제품 발명 예시입니다.

## 파일 구성

- `01_회의록_문제와목적.md`: 기존 작업물 고정 방식의 문제와 발명 목적
- `02_아이디어_구성요소와작동.md`: 레버, 고정축, 잠금부, 조절부 구성
- `03_도면_부호_설명.md`: 도면별 설명과 부호
- `04_선행기술_효과_산업상이용가능성.md`: 비교 키워드, 효과, 적용 분야

## 테스트 포인트

- Markdown 자료도 업로드 자료로 인식되는지 확인
- 필수항목이 대부분 갖춰진 입력에서 질문이 과도하게 나오지 않는지 확인
- Word 초안이 명세서 본문 중심으로 생성되는지 확인

## 권장 메시지

```text
첨부 자료를 분석해서 출원명세서 검토용 초안을 작성해줘.
부족한 항목은 질문으로 알려주고, 근거 없는 수치나 도면부호는 만들지 마.
```
