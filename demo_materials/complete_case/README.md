# 발표용 완성 입력 자료

아래 파일 4개를 SPEC Agent 화면에서 한꺼번에 업로드하세요.

- `01_회의록_문제와목적.md`
- `02_아이디어_구성요소와작동.md`
- `03_도면_부호_설명.md`
- `04_선행기술_효과_산업상이용가능성.md`

권장 메시지:

```text
첨부 자료를 모두 분석해서 출원명세서 검토용 초안을 작성해줘.
부족한 항목이 있으면 체크리스트와 추가 질문으로 분리해서 보여줘.
근거 없는 실험 결과나 수치는 만들지 말고, 자료에 있는 내용만 반영해줘.
```

목표:

- 발명 명칭
- 기술분야
- 배경기술/종래 문제
- 해결하려는 과제
- 구성요소와 해결수단
- 작동 방식/실시예
- 효과와 근거
- 도면 설명/부호
- 선행기술 키워드/문헌
- 산업상 이용가능성

위 10개 체크리스트가 모두 통과하도록 구성했습니다.
