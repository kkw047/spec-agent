# 층상 식품 성형체

서로 다른 식감과 수분 특성을 가진 층을 조합하는 식품 성형체 발명 예시입니다.

## 파일 구성

- `01_회의녹취록.txt`: 식감 유지와 보관 문제
- `02_아이디어_구성작동.txt`: 층 구조, 차단층, 압착 또는 성형 흐름
- `03_도면부호_설명.txt`: 사시도, 단면도, 제조 공정도 부호
- `04_선행기술_효과_산업상이용가능성.txt`: 비교 키워드, 정성 효과, 적용 분야

## 테스트 포인트

- 식품 재료 후보가 확정 사실이 아니라 검토 항목으로 남는지 확인
- 제조 공정과 효과가 명세서 항목으로 분리되는지 확인
- 수분 차단, 식감 유지 같은 효과가 과장되지 않는지 확인

## 권장 메시지

```text
첨부 자료를 분석해서 식품 성형체 발명의 출원명세서 검토용 초안을 작성해줘.
자료에 없는 재료 배합비나 실험 수치는 만들지 마.
```
