# 슬라이드식 보조받침 유아용 의자

접이식 유아용 의자에 슬라이드식 보조받침부를 넣는 제품 발명 예시입니다.

## 파일 구성

- `01_상담기록_문제와목적.txt`: 기존 유아용 의자의 불편과 발명 목적
- `02_아이디어_구성요소와작동.txt`: 지지부, 좌석부, 받침부, 연결부, 보조받침부의 구성과 작동
- `03_도면_부호_설명.txt`: 도면 설명과 부호 대응
- `04_선행기술_효과_산업상이용가능성.txt`: 비교 키워드, 정성 효과, 적용 분야

## 테스트 포인트

- 상담기록과 도면부호가 같이 있을 때 초안이 얼마나 안정적으로 채워지는지 확인
- “잘 모르겠는데?” 같은 추가 대화가 들어와도 기존 업로드 자료가 유지되는지 확인
- KIPRISPlus 후보가 유아용 의자/받침 구조 중심으로 나오는지 확인

## 권장 메시지

```text
첨부 자료를 분석해서 출원명세서 검토용 초안을 작성해줘.
부족한 항목은 질문으로 알려주고, 자료에 없는 수치나 실험 결과는 만들지 마.
```
