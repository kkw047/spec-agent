# 접이식 덕트팬 비행 자동차

도로 주행과 저고도 비행을 함께 고려한 이동체 발명 예시입니다.

## 파일 구성

- `01_회의록_문제와목적.txt`: 도로 주행/비행 전환의 문제와 목적
- `02_아이디어_구성요소와작동.txt`: 차체, 덕트팬, 전개 링크부, 제어부, 안전부 구성
- `03_도면_부호_설명.txt`: 수납 상태도, 전개 상태도, 제어 블록도 설명
- `04_선행기술_효과_산업상이용가능성.txt`: 비교 키워드, 정성 효과, 적용 분야

## 테스트 포인트

- “하늘을 나는 자동차” 같은 넓은 아이디어도 구체 자료가 있으면 초안화되는지 확인
- 안전장치와 최종 판단이 검토 항목으로 분리되는지 확인
- KIPRISPlus 후보가 자동차/비행/추진 구조 중심으로 검색되는지 확인

## 권장 메시지

```text
첨부 자료를 분석해서 출원명세서 검토용 초안을 작성해줘.
안전성이나 특허성 판단은 확정하지 말고 검토 항목으로 분리해줘.
```
