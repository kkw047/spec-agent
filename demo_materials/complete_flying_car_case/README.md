# 질문 없는 완성형 시나리오 1: 접이식 덕트팬 비행 자동차

아래 파일 4개를 한꺼번에 업로드하세요.

- `01_회의록_문제와목적.txt`
- `02_아이디어_구성요소와작동.txt`
- `03_도면_부호_설명.txt`
- `04_선행기술_효과_산업상이용가능성.txt`

권장 메시지:

```text
첨부 자료를 모두 분석해서 출원명세서 검토용 초안을 작성해줘.
자료에 없는 수치나 실험 결과는 만들지 말고, 사람이 검토해야 할 부분은 검토 항목으로 분리해줘.
```

목표:

- 처음 입력만으로 10개 필수항목이 모두 확인되는 완성형 데모
- “하늘을 나는 자동차”처럼 넓은 아이디어도 구체 정보가 있으면 질문 없이 초안 작성 가능함을 보여주는 데모
- 최종 특허성 판단과 청구범위 확정은 Human-in-the-loop로 남기는 데모
