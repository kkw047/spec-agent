# 자세 보조 인체공학 의자

사용자의 자세나 하중 상태를 감지하고 조절 또는 알림을 제공하는 의자 발명 예시입니다.

## 파일 구성

- `01_회의녹취록.txt`: 자세 불안정 문제와 사용 맥락
- `02_아이디어_구성작동.txt`: 센서부, 조절부, 구동부, 알림부 구성
- `03_도면부호_설명.txt`: 의자 구성도와 제어 흐름 부호
- `04_선행기술_효과_산업상이용가능성.txt`: 비교 키워드, 정성 효과, 적용 분야

## 테스트 포인트

- 센서/모터/알림 같은 구성요소가 해결수단으로 정리되는지 확인
- 임계값이나 실험 수치를 임의 생성하지 않는지 확인
- 부족한 안전성 자료가 검토 항목으로 분리되는지 확인

## 권장 메시지

```text
첨부 자료를 분석해서 인체공학 의자 발명의 출원명세서 검토용 초안을 작성해줘.
자료에 없는 임계값이나 실험 결과는 만들지 마.
```
